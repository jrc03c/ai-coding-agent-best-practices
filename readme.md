# Intro

This repository contains a collection of techniques, tips, and tricks for working with AI coding agents.

# Caveats

**"Best" practices:** While the name of this repo includes "best-practices", the truth is that _best_ practices aren't really known yet with any high degree of certainty. The field is changing on a daily basis, so please think of the recommendations in this repo as merely _good_ practices that work pretty well _right now_.

**Claude Code bias:** I (Josh) mostly only have experience with Claude Code. I want to provide info here that will be generic enough to apply to many different agents, but your mileage may vary.
